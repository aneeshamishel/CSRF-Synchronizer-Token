	1.Download and Install XAMPP
	2.Clone/Download files in the repository

	4.Start Apache and Mysql modules from XAMPP
	5.Open up the browser and type the url: http://localhost/CSRF-Synchronizer-Token
	6.Login credentials
		Username:admin
